# 🎓 Student Management System (GUI)

A premium dark-themed Student Management System built using Python and Tkinter. This application allows users to manage student records efficiently with a modern GUI interface.

---

## 🚀 Features

- ➕ Add student records  
- ✏️ Update student details  
- ❌ Delete student records  
- 📋 View all students in table format  
- 🔍 Search functionality  
- 🖱️ Click on record to auto-fill form  
- 💾 Data stored permanently using JSON file  
- 🌙 Clean and modern dark theme UI  

---

## 🛠️ Technologies Used

- Python  
- Tkinter (GUI)  
- JSON (Data Storage)  

---

## ▶️ How to Run

1. Install Python (if not installed)  
2. Download or clone the repository
